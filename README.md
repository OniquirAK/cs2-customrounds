# cs2-customrounds
A custom round plugin

## Commands
```csharp
# css_voteround - Start voting for the next custom round

# css_{ShortCut} - Start custom round like "css_dr". You can set it in config.

# css_shortlist - Shows all round names and shortcut commands
```
